# FacebookBots
A collection of Python-based automation tools designed to streamline networking tasks on Facebook using Selenium and PyAutoGUI.

FEATURES
  Auto-Friend Requester: Automatically sends friend requests from a specific target list with built-in "human-like" mouse movements and security delay logic.

  Auto-Messenger: Automates sending personalized messages to users, reducing repetitive manual work.

  Anti-Detection Logic: Uses randomized sleep intervals and "Sniper Mode" mouse movements to mimic human behavior and avoid automated flags.

  Remote Debugging: Connects to an existing Chrome session to bypass complex login flows and use active sessions safely.


Prerequisites
  Chrome Debugging Mode: A dedicated Chrome profile for automation.

  Libraries: selenium, pyautogui, webdriver_manager.
